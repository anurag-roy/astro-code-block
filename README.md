# Astro Code Block

Beautiful code blocks for your blog or slides or whatever really, powered by [PrismJS](https://prismjs.com/)

### Features

- Light/Dark mode theme switching (using css variables)
- "Copy to Clipboard" button
- Line numbers
- Line highlighting
